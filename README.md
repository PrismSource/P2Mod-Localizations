# Portal 2: Mod Localizations

A repository for localizations of Portal 2 mods, such as Divinity, Revolution, etc.

[![MIT](https://img.shields.io/badge/license-MIT-blue)](https://github.com/PrismSource/P2Mod-Localizations/blob/main/LICENSE)
![Status](https://img.shields.io/badge/status-active-success)

# Contributing

Any mod is welcome, including Google Translate-based editions. Scripts that improve the workflow are also welcome.

To contribute:

- Fork the repository  
- Create a branch with a clear name (`mod/language`, `tool/xyz`, for example: `Divinity/Polish`)  
- Create a Pull Request  

`.kv3` files can be included, but are not required for mods that do not use P2:CE. Mods using P2:CE must include them.

> [!NOTE]
> For now, translations for languages other than Polish may not be accepted quickly, since I (AbbieInSpace) only speak Polish.  
> Sorry for the inconvenience!

# Resources

For editing files, it is recommended to use [Visual Studio Code](https://code.visualstudio.com/), along with the [Source Engine Support extension](https://marketplace.visualstudio.com/items?itemName=stefan-h-at.source-engine-support), as it enables in-editor compile support for captions.

Any other tools can be found in the `tools` folder.
